1. What is JSP? </br>
    JSP stands for Java Server Pages. It is a server side technology which is used to create dynamic web contents. In JSP, you can include your Java codes into an HTML page. These pages are then compiled into a Servlet by a container.

3. What are the advantages of JSP? </br>
    - Development is faster.
    - Has built-in implicit objects.
    - Can integrate easily with Java.
    - JSP supports MVC (Model View Controller) Pattern

4. What is JSP lifecycle?
    - Translation
    - Compilation
    - Initialization (jspInit())
    - Request Processing (_jspService())
    - Destruction (jspDestroy())
